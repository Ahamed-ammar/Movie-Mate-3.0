# Movie Mate Logo

## How to Add the Logo

1. Save the Movie Mate logo image (the popcorn character) as `movie-mate-logo.png`
2. Place it in this directory: `frontend/public/assets/movie-mate-logo.png`

## Logo Specifications

- **Format**: PNG (with transparent background recommended)
- **Recommended Size**: At least 512x512 pixels for best quality
- **File Name**: `movie-mate-logo.png` (exactly as written)

## Where the Logo Appears

- **Header**: Top-left corner of every page (48px height)
- **Landing Page**: Hero section with bounce animation (128px height)

## Note

The logo file is already referenced in the code. Once you save the image file in this directory, it will automatically appear on your website.
